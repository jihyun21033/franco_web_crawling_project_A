'A하 제조사 KSIC 분류 조사' 자동 웹 크롤링 프로그램입니다.

★★★사용 방법은 아래와 같습니다.★★★

1. 작업할 엑셀 파일의 1행에 컬럼명이 오도록 파일을 반드시 수정합니다.

2. Franko_crawler1 폴더 안의 Franko_crawler1.exe 파일을 실행합니다.

3. 실행된 도스창에서 작업할 엑셀 파일이 있는 경로와 파일명을 확장자명을 포함하여 작성하고 엔터를 누릅니다. ( 반드시 확장자명을 포함해야 합니다.)

4. 작업할 엑셀 파일의 시트명을 작성하고 엔터를 누르면 작업이 시작됩니다.

5. 작업이 끝나면 실행 파일이 있는 폴더 안에 '시트명'_result.xlsx로 결과물이 저장됩니다.

★★★주의사항★★★

1. 엑셀 파일의 양식이 프로그램이 요구하는 양식과 다를 경우 프로그램이 강제종료됩니다. 동봉한 엑셀 파일 예시와 같이 양식을 지켜주세요.

2. 웹 페이지가 다운되거나 인터넷 연결이 중단될 경우 프로그램이 중단됩니다. 네트워크 연결이 불안정한 환경에선 데이터를 분할해서 실행시키는걸 추천드립니다.

3. 서버에 데이터를 요청하고 크롤링해오는 과정에서 다소 시간이 걸립니다. 테스트 당시 한 항목당 평균 5초의 속도를 보였으나 이는 하드웨어, 네트워크, 웹페이지의 환경에 따라 변동이 있을 수 있습니다.

4. 누락된 데이터는 원본 페이지에도 없는 데이터입니다.

5. 실행 파일이 있는 폴더 안에 '시트명'_result.xlsx 파일이 이미 존재할 경우 오류가 발생할 수 있습니다. 기존 파일의 이름을 수정하거나 다른 위치로 옮긴 뒤 프로그램을 실행시켜주세요.

6. Franko_crawler 폴더 안의 실행 파일은 패키지 로딩 속도가 빠르나 실행에 있어 폴더 내부 전체 파일을 요구합니다. 폴더 밖의 실행 파일은 패키지가 압축된 단일 실행 파일로 패키지 로딩 속도가 다소 느리지만 다른 파일을 요구하지 않고 단일 파일로 실행이 가능합니다. 용도에 따라 취사선택하시면 됩니다.
